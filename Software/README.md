# Software
- Programs and instructions that run on hardware, including system software and applications.
- 

## System
- Operating System: core system software that manages hardware resources and provides services for applications.
- Utilities : system programs designed to maintain, manage, protect, and optimize a computer.
- Drivers: low-level software that enables the operating system to communicate with hardware devices.

### Operating system
- Manages CPU and programs
- Manages memory (RAM)
- Manages files and storage
- Controls input and output devices
- Provides a user interface

### Windows
- Developed by Microsoft
- Used on personal computers and laptops
- Easy to use with a graphical interface
- Common in schools, offices, and homes

### Linux
- Open-source operating system
- Used on servers, computers, and embedded systems
- Secure, stable, and customizable
- Popular versions: Ubuntu, Fedora, Debian

### macOS
- Developed by Apple
- Runs only on Apple computers (MacBooks, iMacs)
- Known for stability and design
- Used by designers and developers

### Android
- Mobile operating system by Google
- Used on smartphones and tablets
- Touch-based interface
- Supports many apps from Google Play Store

### iOS
- Developed by Apple
- Used on iPhones and iPads
- Secure and smooth performance

## Apps
- Software designed to perform specific tasks for end-users (e.g., editors, browsers, spreadsheets).
- Type:
    - cloud based: run on internet servers (the cloud) instead of your device.
    - installed: downloaded and stored on device.
